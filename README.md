# Bake & Take Jaunpur

Modern React + Tailwind CSS website for Bake & Take, a bakery and cake shop in Jaunpur, India.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

To test from a phone on the same Wi-Fi:

```bash
npm run dev -- --host 0.0.0.0 --port 5173
```

Then open `http://YOUR_COMPUTER_IP:5173/` on the phone.

## Production Build

```bash
npm run build
```
